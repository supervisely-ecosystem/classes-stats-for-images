## Overview 
Data Exploration for Segmentation and Detection tasks is underestimated by many researchers. The accuracy of your models highly depends on how good you understand data. 

This app generates report with detailed general and per image statistics for all classes in images project. It allows to see big picture as well as shed light щт hidden patterns and edge cases (see <a href="#how-to-use">How to use</a> section).
